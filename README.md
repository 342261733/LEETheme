
# LEETheme - 最帅的轻量级主题管理框架

==============



特性
==============
- 两种设置模式,可根据对象单独添加设置,可通过JSON配置统一设置,满足不同风格的开发者。
- 轻量级设计,整个框架只有2个文件构成。
- 支持所有NSObject子类对象,包含所有常用视图对象的颜色和图片等设置。
- 支持动态添加主题,可实现网络主题切换功能。
- 语法优雅,高效简洁,一行代码完成对象设置。
- 当前主题记忆功能,下一次启动自动布置。
- 异步切换主题样式，避免主线程阻塞。
- 完善的文档注释和使用教程。
